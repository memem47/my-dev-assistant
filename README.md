# my-dev-assistant

My Dev Assistant is a small development support tool for practicing and improving practical software development workflows.

The tool generates development artifacts such as Issue text, branch names, commit messages, and Pull Request descriptions from a short task description.

## Purpose

This project is intended to:

- Build a useful personal development tool
- Practice GitHub-based development workflows
- Learn Issue-driven development
- Practice Conventional Commits
- Improve documentation skills
- Learn how to use ChatGPT and GitHub Copilot in a development process

## Planned Features

The initial version will generate:

- Issue title
- Issue description
- Branch name
- Conventional Commit message
- Pull Request title
- Pull Request description

## Example

Input:

```txt
Add error handling for invalid RAW image size
```

Output:

```txt
Issue title:
Invalid RAW image size error

Branch:
fix/invalid-raw-image-size

Commit:
fix: handle invalid RAW image size

PR title:
Fix invalid RAW image size handling
```

## Development Policy

This project follows a small-step development process:

1. Create an Issue
2. Create a branch
3. Implement or update documentation
4. Commit using Conventional Commits
5. Create a Pull Request
6. Review and merge
